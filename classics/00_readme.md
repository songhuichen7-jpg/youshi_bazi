## 使用方法

1. **查目录**：Glob `**/古籍/[书名]/00_mu-lu.md` → Read 目录文件
2. **找章节**：从目录找到对应章节文件名
3. **Read 章节**：Glob `**/古籍/[书名]/[章节文件名]` → Read
4. **引用**：按 `classical-references.md` 的引用格式输出

**注意**：qiongtong-baojian要逐字读原注（"输情"等深层信息在注释里）；ziping-zhenquan的格局规则是整个分析的地基。
